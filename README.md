## Iniciando com prompt

Como utilizar o prompt de comando.
```
echo Hello World -> Escreve no console: 'Hello World'
dir -> Mostra todo conteúdo do diretório atual
cd [directory] -> Move-se para um outro diretório
. -> Diretório atual
.. -> Diretório anterior
mkdir / md [name] -> Cria um novo diretório
move [file] [directory] -> Move um arquivo para local de sua escolha
type [file] -> Acessar um arquivo
echo [in] > [out] -> Cria um arquivo com os dados de [in]
copy [file] [out] -> Copia um arquivo
rename [file] [out] -> Renomeia um arquivo
del [file] -> Deleta um arquivo
cls -> Limpa o console
```

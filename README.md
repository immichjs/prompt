## Iniciando com prompt

- Como utilizar o prompt de comando.
```
echo Hello World -> Escreve no console: 'Hello World'
dir -> Mostra todo conteúdo do diretório atual
cd [directory] -> Move-se para um outro diretório
mkdir / md [name] -> Cria um novo diretório
move [file] [directory] -> Move um arquivo para local de sua escolha
type [file] -> Exibe o conteúdo de um arquivo
echo [in] > [out] -> Cria um arquivo com os dados de [in]
copy [file] [out] -> Copia um arquivo
rename / ren [file] [out] -> Renomeia um arquivo
del [file] -> Deleta um arquivo
cls -> Limpa o console
set > [out] -> Cria um arquivo
tree -> mostra as pastas e subpastas organizadas em uma árvore
more [file] -> Exibe o conteúdo igual ao type, mas em páginas
rmdir / rd -> Remove diretório
help [command] -> Exibe ajuda do comando no console
```

- Mais informações
```
. -> Diretório atual
.. -> Diretório anterior
/O:S -> Tamanho
/O:D -> Data
> -> Uma única linha
>> -> Se existir um arquivo, adiciona ao final desse arquivo
Espaço -> Para passar de página em página
Enter -> Para passar de linha a linha
Q -> Para sair da exibição do comando
```
